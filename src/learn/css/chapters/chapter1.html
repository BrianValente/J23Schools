¿Qué es CSS?

Característica

  • • CSS significa C ascading S tyle S heets
  • • CSS describe cómo los elementos HTML se van a mostrar en la pantalla, papel, o en otros medios
  • • CSS ahorra mucho trabajo. Se puede controlar el diseño de varias páginas web a la vez
  • • Estilo externas se almacenan en archivos CSS

¿Por qué usarlo?

En una página web CSS se usa para definir estilos, el diseño, la disposición y las variaciones en la imagen para los diferentes tipos de pantallas

CSS y HTML

HTML ofrece un estilo rústico en su diseño de una página web. En cambio, CSS nos brinda una gama amplia de herramientas profesionales con las cuales trabajar.
