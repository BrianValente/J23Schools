CSS Sintaxis y selectores

sintaxis CSS

Un conjunto de reglas CSS consta de un selector y un bloque de declaración:

El bloque de declaración contiene una o más declaraciones separadas por punto y coma

Cada declaración incluye un nombre de propiedad CSS y un valor, separados por dos puntos

Una declaración CSS siempre termina con un punto y coma, y los bloques de declaraciones están rodeados por llaves

En el siguiente ejemplo todos los elementos <p> estarán alineados-centro, con un color rojo texto:

p { color : green; text-align: center; }

Los selectores CCS

se utilizan para "encontrar" (o seleccionar) elementos HTML en función de su nombre de elemento, de la identificación, clase, atributo, y mucho más

El selector de elemento

Selecciona elementos basados en el nombre del mismo

Puede seleccionar todos los elementos <p> en una página como esta (en este caso, todos los elementos <p> estarán alineados-centro, con un color rojo texto):

Ejemplo:

p { text-align: center; color: red; }

El selector ID

Utiliza el atributo id de un elemento HTML para seleccionar un elemento específico

El selector ID debe ser único, ya que deberá buscar el elemento especifico de la página, y no debe confundirse con otros

Para seleccionar un elemento con un ID específico, escribe un carácter hashtag (#), seguido por el nombre de id del elemento

La regla de estilo a continuación se aplica al elemento HTML con id = "párrafo1":

#para1 { text-align: center; color: red; }

Nota: Una ID no puede comenzar con un número!

El selector de clase

Selecciona los elementos con un atributo de clase específica

Para seleccionar elementos con una clase específica, escriba un punto (.) carácter, seguido del nombre de la clase

En el siguiente ejemplo, todos los elementos HTML con class = "centro" será de color rojo y alineado al centro:

.center { text-align: center; color: green; }

Los elementos HTML especificados solamente deben verse afectados por una clase

En el siguiente ejemplo, solamente <p> elementos con class = "centro" estarán alineados-centro:

p.center { text-align: center; color: green; }
