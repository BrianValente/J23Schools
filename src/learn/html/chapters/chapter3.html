Capítulo 3: Ejemplos basicos


Los documentos HTML

Todos los documentos HTML deben comenzar con una declaracion de tipo de documento:<!Doctype html>

El documento HTML en si comienza con <html> y termina con </html>. La parte visible del documento HTML esta entre <body> y </body>

Ejemplo:

<html>
<head>
<title>Titulo de la pagina</title>
</head>
<body>

</body>
</html>

Encabezamientos HTML

Los titulos HTML se definen desde la etiqueta <h1> a <h6>

<h1> es el titulo mas importante. <H6> titulo menos importante

Ejemplo

<html>
<head>
<title>Titulo de la pagina</title>
</head>
<body>
<h1> Encabezado tipo 1 </h1>
<h2> Encabezado tipo 2 </h2>
</body>
</html>

Los Parrafos HTML

En HTML los parrafos se definen con la etiqueta<p>


Ejemplo:

<html>
<head>
<title>Titulo de la pagina</title>
</head>
<body>
<p> ¡Mi primer parrafo! </p>
</body>
</html>
