Capítulo 8: Estilos


Atributo "Style"

Para agregar estilos en un elemento, se lo tiene que hacer por el atributo "Style"

Sintaxis:

<html>
<head>
<title>Titulo de la pagina</title>
</head>
<body>
<nombredeetiqueta style="propiedad:valor;">
</body>
</html>
La propiedad es una propiedad de CSS, y el valor es de CSS también.

Color de texto

La propiedad "color" define el color de texto de un elemento. Por ejemplo este parrafo.

Sintaxis:

<html>
<head>
<title>Titulo de la pagina</title>
</head>
<body>

</body>

Fuentes de HTML

La propiedad "font-family" define el tipo de fuente que se va a utilizar en un elemento. Por ejemplo en este parrafo.

Sintaxis:

<html>
<head>
<title>Titulo de la pagina</title>
</head>
<body>
<p style="font-family:courier;"> Este parrafo esta con la fuente courier </p>
</body>
</html>

Tamaño de texto

La propiedad "font-size" define el tamaño de texto de un elemento HTML. Por ejemplo este parrafo.

Sintaxis:

<head>
<title>Titulo de la pagina</title>
</head>
<body>
<p style="font-size:150%;"> Este parrafo tiene un tamaño de letra mayor </p>
</body>
</html>

Alineacion de texto

La propiedad "text-align" define en que alineacion queremos que se presente un elemento. Por ejemplo este parrafo.

Sintaxis:

</head>
