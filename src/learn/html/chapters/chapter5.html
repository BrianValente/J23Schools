Capítulo 5: Atributos HTML


  • Todos los elementos HTML pueden tener atributos
  • Los atributos proveen información adicional acerca de un elemento
  • Los atributos siempre están escritos en la etiqueta de apertura
  • Los atributos normalmente vienen en forma nombre/valor. Por ejemplo: nombre_atributo = "valor"

Atributo lang

El idioma de este documento puede ser declarado dentro de la etiqueta <html>.

El idioma es definido mediante el atributo lang.

Declarar un idioma es importante para las aplicaciones de accesibilidad y los motores de búsqueda

<html lang="es">

Quiere decir que el idioma definido para el documento es el español (es).

No olvides que todos los valores de los atributos se encuentran encerrados entre comillas


Atributo title

El atributo title se define dentro de una etiqueta <p>. Lo que diga dentro de este atributo será mostrado cuando pases el mouse por encima del párrafo en cuestión.


Atributo href

Los hipervínculos (también llamados links) se definen dentro de la etiqueta <a>. La dirección del hipervínculo está especificada dentro del atributo href.


Atributos de tamaño

Las imágenes en HTML se definen mediante la etiqueta <img>.

La ubicación de la imagen (src) y su tamaño, tanto ancho (width) como altura (height) están en forma de atributos

Atributo alt

El atributo alt especifica un texto alternativo para ser mostrado en caso de que no llegue a cargarse una imagen.
