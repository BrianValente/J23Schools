Capítulo 4: Elementos HTML

Un elemento HTML generalmente consiste en una etiqueta de inicio y otra de fin, con el contenido encontrándose entre estos dos


<etiqueta > Contenido </etiqueta>


El elemento HTML es todo desde la primera etiqueta hasta el final


<p> Un párrafo </p>


Etiqueta de comienzo Contenido del elemento Etiqueta de fin
<h1> Mi primer encabezado </h1>
<p> Mi primer encabezado </p>
<br>

Los elementos HTML pueden estar anidados, es decir, pueden haber elementos dentro de un elemento, y ese elemento encontrarse dentro de otro más.

Veamos el clásico ejemplo:

<!DOCTYPE html> <html> <body> <h1>Un encabezado</h1> <p>Un párrafo.</p> </body> </html>

El elemento <html> define el documento entero, y dentro de él se encuentra el cuerpo, el elemento <body>. Y a su vez, dentro de éste se encuentran el encabezado (<h1>) y el párrafo (<p>).
