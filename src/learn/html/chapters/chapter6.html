Capítulo 6: Encabezados

Los encabezados se definen con las etiquetas <h1> hasta <h6>.

<h1> define los encabezados mas importantes, y los de <h6> los de menor importancia.

<html>
<head>
<title>Titulo de la pagina</title>
</head>
<body>
<h1>Esto es un encabezado de tipo H1</h1>
<h2>Esto es un encabezado de tipo H2</h2>
<h3>Esto es un encabezado de tipo H3</h3>
<h4>Esto es un encabezado de tipo H4</h4>
<h5>Esto es un encabezado de tipo H5</h5>
<h6>Esto es un encabezado de tipo H6</h6>
</body>
</html>

Reglas horizontales

La etiqueta <hr> define una separacion de tema en una pagina HTML, y se la muestra como una linea horizontal. Se usa para separar contenido de la pagina. Ejemplo:

Sintaxis:

<html>
<head>
<title>Titulo de la pagina</title>
</head>
<body>
<p> Hola mundo! </p>
<hr>
</body>
</html>

El elemento <head> de HTML

En este elemento introduciremos las etiquetas que no influyen en contenido a la pagina, por ejemplo la etiqueta <title>. Esta etiqueta se posiciona despues de la etiqueta <html> y <body>

<html>
<head>

</head>
<body>

</body>
</html>
Principalmente se definen: Titulos del documento,conjunto de caracteres, estilos, links, scripts y otro tipos de informacion.
