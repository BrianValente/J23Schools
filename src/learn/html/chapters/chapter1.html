Capítulo 1: Introducción a HTML

¿Qué es HTML?

HTML es el lenguaje estándar de etiquetas usado para crear páginas Web.

HTML significa: Hyper Text Markup Language (Lenguaje de Marcas de HiperTexto)

Los elementos HTML son el bloque de construcción de una página HTML. Estos elementos están representados mediante "etiquetas".

El navegador web no mostrará estas etiquetas, pero las utilizará para mostrar el contenido de la página

Un ejemplo de HTML

Un pequeño documento HTML:

<!DOCTYPE html>
<html>
	<head>
		<title>Titulo de pagina</title>
	</head>
	<body>

		<h1>Mi primer encabezado.</h1>
		<p>Mi primer párrafo.</p>

	</body>
</html>

Explicación del ejemplo

  • La declaración !DOCTYPE html define que el tipo de documento es HTML
  • Todo el contenido de este documento HTML va entre las etiquetas <html> y </html>
  • El texto entre <head> y </head> da información sobre el documento en general, no será mostrado en el navegador
  • El texto entre <title> y </title> le da un título a la página.
  • Todo el contenido de la página que se va a ver dentro del navegador se encuentra entre <body> y </body>
  • El texto entre <h1> y </h1> define un encabezado
  • El texto entre <p> y </p> define un párrafo

Dentro del navegador web, este documento se mostrará como una simple página con un encabezado y un párrafo.

Etiquetas HTML

Como podemos ver, las etiquetas HTML son "palabras clave" encerradas dentro de corchetes angulares:

<etiqueta>contenido</etiqueta>
  • Estas etiquetas HTML normalmente vienen en pares, como de esta forma: <p> y </p>
  • La primera etiqueta es la etiqueta de comienzo, la segunda es la etiqueta de fin.
  • La etiqueta final se escribe como la primera, pero con una barra (/) antes del nombre de dicha etiqueta.
La etiqueta del comienzo se llama "etiqueta de apertura", y la final se la llama "etiqueta de cierre".

Estructura de página HTML

Debajo hay una visualizacion de la estructura de una pagina HTMl:

<html>
<head>
<title>Titulo de la pagina</title>
</head>
<body>
<h1>Esto es un encabezado.</h1>
<p>Esto es un parrafo.</p>
<p>Esto es otro parrafo.</p>
</body>
</html>
Sólo el area dentro de la etiqueta "<body>" se muestra en el navegador web.

La declaración <!DOCTYPE>:

La declaración <!DOCTYPE> ayuda al navegador a mostrar la página web correctamente. Hay diferentes tipos de documentos en la web, y por lo tanto, para mostrar un documento de forma correcta, el navegador debe saber tipo y versión del documento. La declaración <!DOCTYPE> no distingue las mayúsculas de las minúsculas. Todas las formas son aceptables.:

<!DOCTYPE html> <!DOCTYPE HTML> <!doctype html> <!Doctype Html>
