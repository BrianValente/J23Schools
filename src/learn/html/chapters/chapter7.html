Capítulo 7: Párrafos

Se usa <p> para definir un párrafo.

<p>Este es un párrafo.</p> <p>Este es otro párrafo.</p>

Visualización

No se puede saber con seguridad cómo un documento HTML será mostrado en el navegador

Grandes o pequeñas pantallas, ventanas de tamaños diferentes, crearán diferentes resultados.

Con HTML, no se puede cambiar lo que se muestra simplemente poniendo espacios libres o saltos de línea (renglón) en el código.

El navegador removerá cualquier espacio extra y cualquier línea de más al momento de mostrar la página. Por ejemplo:

<p> Este párrafo contiene varias líneas en el código fuente, pero el navegador lo ignora. </p>
<p> Este párrafo contiene      un montón de espacios en el código      fuente, pero el      navegador lo ignora. </p>

No olvides la etiqueta final

La mayoría de los navegadores mostrarán el HTML correctamente incluso aunque te olvides de poner la etiqueta final (en este caso, </p>).

<p>Un párrafo. <p>Otro párrafo.

Este ejemplo se mostrará correctamente en la mayor parte de los navegadores, pero no te acostumbres. Poné siempre la etiqueta final: </p>.


Saltos de línea

El elemento <br> define un salto de línea (un nuevo renglón).

Usá <br> si querés un salto de línea sin tener que empezar un nuevo párrafo.

<p>Este es<br>un párrafo<br>con saltos de línea.<p>

La etiqueta <br> es una etiqueta vacía, es decir que no tiene etiqueta final, no existe un "</br>".


El elemento <pre>

El elemento <pre> define un texto con formato predefinido.

Un texto dentro de <pre> es mostrado en una fuente de ancho fijo (normalmente Courier), y preserva tanto espacios como saltos de línea.

  <pre>
  Este texto            tiene varios espacios
  y también           varios saltos
  de línea,           y el explorador
  lo mostrará así           como está ahora.
  </pre>
