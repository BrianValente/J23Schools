Capítulo 2: Editores de HTML


Programar en HTML con un bloc de notas:

Las paginas web pueden ser desarrolladas y modificadas usando un editor profesional de HTML.

Igual recomendamos que se utilice el bloc de notas (En el caso de MAC, Textedit).

Creemos que usar este simple bloc de notas es una buena forma de aprender HTML.

Para crear nuestro archivo HTML, te recomendamos seguir estos pasos:

Paso 1: Abrir el Bloc de notas

De Windows 8 o posterior:

Abra la pantalla de inicio (el simbolo ventana en la parte inferior izquierda de la pantalla). Escriba el Bloc de notas .

Windows 7 o una version anterior:

Abrir en Inicio > Programas> Accesorios> Bloc de notas

Paso 1: Abra TextEdit (Mac)

Abra Finder> Aplicaciones> TextEdit

Tambien cambiar algunas preferencias para conseguir la aplicacion para guardar los archivos correctamente. En Preferencias> Formato> elegir "Texto sin formato"

A continuacion, en "Abrir y guardar", marque la casilla que dice "Ignorar comandos de texto enriquecido en los archivos HTML".

A continuacion, abra un nuevo documento para colocar el codigo.

Paso 2: Escribir un HTML

Escribir o copiar algo de HTML en el Bloc de notas.

<!DOCTYPE HTML> <html> <body> <First Heading> <My first paragraph.> </body> </html>

Paso 3: Guardar la pagina HTML

Guarde el archivo en su ordenador. Seleccione Archivo> Guardar como en el menu Bloc de notas.

Nombre del archivo "index.htm" y establecer la codificacion a UTF-8 (que es la codificacion preferida para archivos HTML).




Puede utilizar cualquiera de .htm o .html como extension de archivo. No hay ninguna diferencia, que depende de usted.


Paso 4: Ver la pagina HTML en su navegador

Abra el archivo HTML guardado en su navegador favorito (doble clic sobre el archivo, o haga clic derecho - y elegir la opcion "Abrir con").
