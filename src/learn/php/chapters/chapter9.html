Los operadores de PHP

Los operadores se utilizan para realizar operaciones sobre las variables y valores. PHP divide a los operadores en los siguientes grupos:

  • Operadores aritméticos
  • Operadores de Asignación
  • Los operadores de comparación
  • operadores de incremento / decremento
  • Operadores logicos
  • operadores de cadena
  • operadores de matriz

Los operadores aritméticos PHP

Los operadores aritméticos se usan PHP con valores numéricos para realizar operaciones aritméticas comunes, tales como suma, resta, multiplicación, etc.

Operador Nombre Ejemplo Resultado
+ Suma $x + $y Suma $x y $y
- Resta $x - $y resta $x y $y
* Multiplicación $x * $y multiplica $x y $y
/ División $x / $y divide $x y $y
% Porcentaje $x % $y Porcentaje de $x por $y
** Potencia $x ** $y potenciacia a $x por $y

Operadores de comparación de PHP

Los operadores de comparación PHP se utilizan para comparar dos valores (número o cadena):

Operador Nombre Ejemplo Resultado
== Igual $x == $y devuelve si $x es igual a $y
=== Identico $x === $y devuelve true si $x es igual a $y, y son del mismo tipo
!= No igual $x != $y devuelve true si $x no es igual a $y
<> No igual $x <> $y devuelve true si $x no es igual a $y
!== No idénticos $x !== $y devuelve true si $x no es igual a $y, o si no son del mismo tipo
> Mayor $x > $y devueltve true si $x es más grande que $y
< Menor $x < $y devuelve true si $x es más chico que $y
>= Mayor o igual $x >= $y devuelve true si $x es mayor o igual que $y
<= Menor o igual $x <= $y devuelve true si $x es menor o igual que $y

Operadores de incremento y decremento

Operador Nombre Descripción
++$x Pre-incrementar Incrementa $x en uno, después devuelve $x
$x++ Post-incrementar Devuelve $x, después incrementa $x en uno
--$x Pre-decrementar Decrementa $x en uno, después devuelve $x
$x-- Post-decrementar Devuelve $x, después decrementa $x por 1

Operadores lógicos

Operador Nombre Ejemplo Resultado
and And $x and $y Devuelve true si $x y $y son true
or Or $x or $y Devuelve true si $x o $y es true
xor Xor $x xor $y True if either $x or $y is true, but not both
&& And $x && $y True if both $x and $y are true
|| Or $x || $y True if either $x or $y is true
! Not !$x True if $x is not true
