Php 5 incluir archivos

La declaracion incluir lleva todo el texto / código / marcado que existe en el archivo y lo copia especificada en el archivo que utiliza la instrucción de inclusión. Incluir archivos es muy útil cuando se desea incluir el mismo PHP, HTML o texto en varias páginas de un sitio web.

PHP incluye y requiere de Declaraciones

Es posible insertar el contenido de un archivo PHP en otro archivo PHP (antes de que el servidor lo ejecute), con la declaración de incluir o requerir. Las sentencias de inclusión y requieren son idénticos, excepto en caso de fallo:

  • Requerir producirá un error fatal (E_COMPILE_ERROR) y detener el script
  • Incluir sólo producirá una advertencia (E_WARNING) y el guión seguirá

    • Por lo tanto, si desea que la ejecución de seguir y mostrar a los usuarios la salida, incluso si el archivo de inclusión no se encuentra, usar la instrucción de inclusión. De lo contrario, en el caso de Marco, CMS, o una codificación de aplicaciones PHP compleja, siempre utilice la instrucción requiere para incluir un archivo de clave para el flujo de ejecución. Esto ayudará a evitar poner en peligro la seguridad y la integridad de su aplicación, por si a caso un archivo clave es accidentalmente falta. Incluyendo archivos ahorra mucho trabajo. Esto significa que puede crear un encabezado, pie de página o archivo de menú estándar para todas sus páginas web. Entonces, cuando el encabezado necesita ser actualizado, sólo se puede actualizar el encabezado de archivo de inclusión.

    Sintaxis

    include 'nombredelarchivo'; or require 'nombredelarchivo';

    Ejemplos de Incluir en PHP

    Ejemplo 1

    Supongamos que tenemos un archivo de pie de página estándar llamado "pie.php", que se ve así:

    <?php echo "<p>Copyright © 1999-" . date("Y") . " J23schools.com</p>"; ?>

    Para incluir el archivo de pie de página en una página, utilice la instrucción de inclusión:

    Ejemplo

    <html> <body> <h1>Bienvenido a mi pagina principal! <p>Algo de texto.

    <p>Un poco mas de texto.

    <?php include 'footer.php'?> </body> </html>;

    Ejemplo 2

    Supongamos que tenemos un archivo de menú estándar llamado "menu.php":

    <?php echo '<a href="/default.asp">Home</a> - <a href="/html/default.asp">HTML Tutorial</a> - <a href="/css/default.asp">CSS Tutorial</a> - <a href="/js/default.asp">JavaScript Tutorial</a> - <a href="default.asp">PHP Tutorial</a>' ?>;

    Ejemplo 3

    Supongamos que tenemos un archivo llamado "vars.php", con algunas variables definidas:

    p

    Entonces, si incluimos el archivo "vars.php", las variables se pueden utilizar en el archivo de llamada:

    Ejemplo

    <html> <body> <h1>Welcome to my home page!</h1> <?php include 'vars.php'; echo "Tengo un $auto $color."; ?> </body> </html>

    PHP incluir vs. requierir

    La declaración requerir también se utiliza para incluir un archivo en el código PHP. Sin embargo, hay una gran diferencia entre incluir y requerir; cuando un archivo se incluye con la declaración incluir y PHP no puede encontrarlo, el guión continuará ejecutando:

    Ejemplo

    <html> <body> <h1>Bienvenido a mi pagina principal!</h1> <?php include 'noexisteelarchivo.php'; echo "Tengo un $auto $color."; ?> </body> </html>
