Matrices multidimensionales

Una matriz multidimensional es una matriz que contiene una o más matrices. PHP entiende matrices multidimensionales que son dos, tres, cuatro, cinco, o más niveles de profundidad. Sin embargo, las matrices más de tres niveles de profundidad son difíciles de manejar para la mayoría de la gente.

La dimensión de una matriz indica el número de índices que necesita para seleccionar un elemento. Para una matriz bidimensional que necesita dos índices para seleccionar un elemento Para una matriz tridimensional se necesitan tres índices para seleccionar un elemento

Arrays bidimensionales

Una matriz bidimensional es una matriz de matrices (una matriz tridimensional es una matriz de matrices de matrices). En primer lugar, echar un vistazo a la siguiente tabla:

Nombre Stock Vendidos
Volvo 22 18
BMW 15 13
Saab 5 2
Land Rover 17 15

Podemos almacenar los datos de la tabla anterior en una matriz de dos dimensiones, como esto: $cars = array ( array("Volvo",22,18), array("BMW",15,13), array("Saab",5,2), array("Land Rover",17,15) ); Ahora el bidimensional $ coches matriz contiene cuatro matrices, y tiene dos índices: fila y columna. Para obtener acceso a los elementos de la matriz $ coches hay que apuntar a los dos índices (fila y columna):

<?php echo $cars[0][0].": In stock: ".$cars[0][1].", sold: ".$cars[0][2].".<br>"; echo $cars[1][0].": In stock: ".$cars[1][1].", sold: ".$cars[1][2].".<br>"; echo $cars[2][0].": In stock: ".$cars[2][1].", sold: ".$cars[2][2].".<br>"; echo $cars[3][0].": In stock: ".$cars[3][1].", sold: ".$cars[3][2].".<br>"; ?>

También podemos poner un bucle dentro de otro bucle For para obtener los elementos de la matriz $ coches (que aún tenemos que apuntar a los dos índices):

<?php for ($row = 0; $row < 4; $row++) { echo "<p><b>Row number $row</b></p>"; echo "<ul>"; for ($col = 0; $col < 3; $col++) { echo "<li>".$cars[$row][$col]."</li>"; } echo "</ul>"; } ?>
