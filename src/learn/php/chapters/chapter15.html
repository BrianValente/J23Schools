Array

Una matriz almacena varios valores en una sola variable:

<?php $cars = array("Volvo", "BMW", "Toyota"); echo "I like " . $cars[0] . ", " . $cars[1] . " and " . $cars[2] . "."; ?>

¿Qué es una matriz?

Una matriz es una variable especial, que puede contener más de un valor a la vez. Si usted tiene una lista de elementos (una lista de nombres de coche, por ejemplo), el almacenamiento de los coches en las variables individuales podría tener este aspecto: $cars1 = "Volvo"; $cars2 = "BMW"; $cars3 = "Toyota"; Sin embargo, lo que si desea colocar a través de los coches y encontrar una específica? ¿Y si no tenías 3 coches, pero 300? La solución es crear una matriz! Una matriz puede contener muchos valores bajo un único nombre, y se puede acceder a los valores de referencia a un número de índice.

Crear un array en PHP

En PHP, la función array () se utiliza para crear una matriz: array(); En PHP, hay tres tipos de matrices:

  • Arrays indexados - cadenas con un índice numérico
  • Los array asociativos - cadenas con claves denominadas
  • Los array multidimensionales - Las matrices que contienen una o más matrices

Arrays indexados

Hay dos maneras de crear matrices indexadas: El índice se puede asignar de forma automática (índice siempre empieza en 0), así: $cars = array("Volvo", "BMW", "Toyota"); o el índice se puede asignar de forma manual: $cars[0] = "Volvo"; $cars[1] = "BMW"; $cars[2] = "Toyota"; En el siguiente ejemplo se crea una matriz indexada llamada $ coches, asigna tres elementos a la misma, y ​​luego imprime un texto que contiene los valores de la matriz:

<?php $cars = array("Volvo", "BMW", "Toyota"); echo "I like " . $cars[0] . ", " . $cars[1] . " and " . $cars[2] . "."; ?>

Obtener la longitud de un array

La función count () se utiliza para devolver la longitud (el número de elementos) de un array:

<?php $cars = array("Volvo", "BMW", "Toyota"); echo count($cars); ?>

Array asociativos

Las matrices asociativas son matrices que utilizan claves con nombre que se asignan a los mismos. Hay dos maneras de crear una matriz asociativa: $age = array("Peter"=>"35", "Ben"=>"37", "Joe"=>"43"); o: $age['Peter'] = "35"; $age['Ben'] = "37"; $age['Joe'] = "43"; Las teclas nombrados a continuación, se pueden utilizar en una secuencia de comandos:

<?php $age = array("Peter"=>"35", "Ben"=>"37", "Joe"=>"43"); echo "Peter is " . $age['Peter'] . " years old."; ?>

Bucle de array asociativo

Para recorrer e imprimir todos los valores de una matriz asociativa, se puede utilizar un bucle foreach, como esto:

<?php $age = array("Peter"=>"35", "Ben"=>"37", "Joe"=>"43"); foreach($age as $x => $x_value) { echo "Key=" . $x . ", Value=" . $x_value; echo "<br>"; } ?>
