Mantener los valores en el formulario

Para mostrar los valores en los campos de entrada después de que el usuario pulsa el botón de enviar, añadimos un pequeño script PHP dentro del atributo de valor de los siguientes campos de entrada: nombre, correo electrónico y página web. En el campo de comentarios área de texto, ponemos la secuencia de comandos entre el Gender: <input type="radio" name="gender" <?php if (isset($gender) && $gender=="female") echo "checked";?> value="female">Female <input type="radio" name="gender" <?php if (isset($gender) && $gender=="male") echo "checked";?> value="male">Male
