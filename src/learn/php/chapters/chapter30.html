PHP 5 Galletas

Una cookie o galleta se utiliza a menudo para identificar a un usuario.

¿Qué es una cookie?

Una cookie se utiliza a menudo para identificar a un usuario. Una cookie es un pequeño archivo que el servidor se incrusta en el ordenador del usuario./
Cada vez que el mismo equipo solicita una página con un navegador, se enviará la cookie también. Con PHP, puede tanto crear y recuperar los valores de las cookies.

Crear Galletas Con PHP

Una cookie es creado con la función setcookie ().

Sintaxis

setcookie(nombre, valor, expiracion, direccion, dominio, seguridad, httponly);

Sólo el nombre es un requerido . Todos los demás parámetros son opcionales.

PHP Crear/Recuperar una galleta

En el siguiente ejemplo se crea una cookie llamada "usuario" con el valor "John Doe". La cookie caducará después de 30 días (86400 * 30). El "/" significa que la cookie está disponible en todo el sitio web (en caso contrario, seleccionar el directorio que prefiera). A continuación, recuperar el valor de la cookie "usuario" (usando la variable global $ _COOKIE). También usamos la función isset () para averiguar si se ha establecido la cookie:

Ejemplo

<html> <body> <?php if(!isset($_COOKIE[$cookie_name])) { echo "La galleta llamada '" . $cookie_name . "' no esta colocada!"; } else { echo "La galleta '" . $cookie_name . "' esta colocada!
"; echo "El valor es: " . $_COOKIE[$cookie_name]; } ?> </body> </html>

Nota: La función setcookie () debe aparecer antes de la etiqueta .

Nota: El valor de la cookie se urlencoded automáticamente cuando se envía la cookie, y decodifica automáticamente cuando se reciben (para evitar URLEncoding, el uso setrawcookie () en su lugar).

Modificar un valor de cookie o galleta

Para modificar una cookie, acaba de establecer (de nuevo) la cookie utilizando la función setcookie ():

Ejemplo

<?php $cookie_name = "user"; $cookie_value = "Alex Porter"; setcookie($cookie_name, $cookie_value, time() + (86400 * 30), "/"); ?> <html> <body> <?php if(!isset($_COOKIE[$cookie_name])) { echo "La cookie llamada'" . $cookie_name . "' no esta colocada!"; } else { echo "La cookie '" . $cookie_name . "' esta colocada!
"; echo "Su valor es: " . $_COOKIE[$cookie_name]; } ?> </body> </html>

Eliminar una galleta

Para eliminar una cookie, utilice la función setcookie () con una fecha de caducidad en el pasado:

Ejemplo

<?php // Coloca la hora de expiracion de la cookie para hace una hora setcookie("user", "", time() - 3600); ?> <html> <body> <?php echo "La galleta de'user' fue borrada."; ?> </body> </html>

Compruebe si están activadas las cookies

En el siguiente ejemplo se crea un pequeño script que comprueba si las cookies están habilitadas. En primer lugar, tratar de crear una cookie de prueba con la función setcookie (), y luego contar la variable de matriz $ _COOKIE:

Ejemplo

<?php setcookie("test_cookie", "test", time() + 3600, '/'); ?> <html> <body> <?php if(count($_COOKIE) > 0) { echo "Las cookies estan activadas."; } else { echo "Las cookies estan desactivadas."; } ?> </body> </html>
