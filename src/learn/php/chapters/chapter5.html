PHP 5 echo y declaraciones de impresión

En PHP hay dos formas básicas para obtener una salida: eco y de impresión. En este tutorial usamos eco (e imprimir) en casi todos los ejemplos. Por lo tanto, este capítulo contiene un poco más de información acerca de esos dos estados de salida.

echo y print

echo y print son más o menos lo mismo. Ambos se utilizan para enviar datos a la pantalla. Las diferencias son pequeñas: echo no tiene valor de retorno, mientras que print tiene un valor de retorno de 1 por lo que se puede utilizar en expresiones. echo puede tomar varios parámetros (aunque tal uso es poco frecuente), mientras que print puede tener un argumento. echo es marginalmente más rápido que print.

Declarar echo en PHP

La declaración de eco se puede utilizar con o sin paréntesis: echo o echo ().

El siguiente ejemplo muestra cómo la salida de texto con el comando echo (aviso de que el texto puede contener marcado HTML):

<?php echo "<h2>PHP es divertido!</h2>"; echo "Estoy aprendiendo PHP!<br>"; echo "este","string","fue","hecho","con multiples parámetros"; ?>

Print en PHP

La sentencia print se puede utilizar con o sin paréntesis:print o print().

El siguiente ejemplo muestra cómo el texto de salida con el comando de impresión (aviso de que el texto puede contener marcado HTML):

<?php print "Hello world!"; print "<h2>PHP es divertido!</h2>"; print "Estoy aprendiendo PHP!<br>"; ?>

Las variables de visualización

El siguiente ejemplo muestra cómo el texto de salida y las variables con la declaración de impresión:

<?php $txt1 = "Learn PHP"; $txt2 = "W3Schools.com"; $x = 5; $y = 4; print "<h2>$txt1</h2>"; print "Study PHP at $txt2<br>"; print $x + $y; ?>
