For

El bucle se utiliza cuando se sabe de antemano cuántas veces la secuencia de comandos debe ejecutarse.

Sintaxis

for (init counter; test counter; increment counter) { codigo a ser ejecutado; } parámetros:

  • init counter : Inicializa el valor de contador de bucle
  • test counter : evalúa para cada iteración del bucle. Si se evalúa como TRUE, el bucle continúa. Si se evalúa como FALSE, el bucle termina.
  • increment counter : Aumenta el valor del contador de bucle
El ejemplo a continuación muestra los números del 0 al 10:

<?php for ($x = 0; $x <= 10; $x++) { echo "The number is: $x
"; } ?>

Foreach

El bucle foreach funciona solamente con matrices, y se utiliza para recorrer cada par clave / valor en una matriz.

Sintaxis

foreach ($array as $value) { codigo a ser ejecutado; } Para cada iteración del bucle, el valor del elemento de matriz actual se asigna al valor de $ y el puntero se mueve a una, hasta que alcanza el último elemento de la matriz. El siguiente ejemplo muestra un bucle que dará salida a los valores de la matriz dada ($) colores:

<?php $colors = array("red", "green", "blue", "yellow"); foreach ($colors as $value) { echo "$value <br>"; } ?>
