PHP Filtros avanzados

Validar un número entero dentro de un rango

En el siguiente ejemplo se utiliza la función filter_var () para comprobar si una variable es tanto de tipo INT, y entre 1 y 200:

Ejemplo

<?php $int = 122; $min = 1; $max = 200; if (filter_var($int, FILTER_VALIDATE_INT, array("options" => array("min_range"=>$min, "max_range"=>$max))) === false) { echo("El valor de la variable no se encuentra en un rango legal"); } else { echo("El valor de la variable se encuentra en un rango legal"); } ?>

Validar direcciones IPv6

En el siguiente ejemplo se utiliza la función filter_var () para comprobar si el $ ip variable es una dirección IPv6 válida:

Ejemplo

<?php $ip = "2001:0db8:85a3:08d3:1319:8a2e:0370:7334"; if (!filter_var($ip, FILTER_VALIDATE_IP, FILTER_FLAG_IPV6) === false) { echo("$ip es una direccion IPv6 valida"); } else { echo("$ip no es una direccion IPv6 valida"); } ?>

Validar URL - debe contener una cadena de consulta

En el siguiente ejemplo se utiliza la función filter_var () para comprobar si el $ url variable es una URL con una cadena de consulta:

Ejemplo

<?php $url = "http://www.j23schools.com"; if (!filter_var($url, FILTER_VALIDATE_URL, FILTER_FLAG_QUERY_REQUIRED) === false) { echo("$url es una URL valida"); } else { echo("$url no es una URL valida"); } ?>
