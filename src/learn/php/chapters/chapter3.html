Sintaxis básica de PHP

Un script PHP se puede colocar en cualquier parte del documento. Un script PHP comienza con : <?php ?>

La extensión de archivo por defecto para archivos PHP es ".php". Un archivo PHP normalmente contiene etiquetas HTML, y algo de código de programación PHP. A continuación, tenemos un ejemplo de un archivo PHP simple, con un script PHP que utiliza una función integrada de PHP "eco" a la salida del texto "Hello World!" en una página web:

Ejemplo

<!DOCTYPE html> <html> <body> <?php

// Este es un comentario de una sola línea

#Este también es un comentario de una sola línea

/* Este es un comentario que sirve para abarcar varias lineas */

// También se puede usar para comentar partes del código

$x = 5/* + 15 */ + 5; echo $x; ?> </body> </html>

Sensibilidad a las mayúsculas

En PHP, todas las palabras clave (por ejemplo, si, no puede, cuando, eco, etc.), clases, funciones y funciones definidas por el usuario no entre mayúsculas y minúsculas. En el ejemplo siguiente, los tres estados de eco a continuación son legales (e igual):

Ejemplo

<!DOCTYPE html> <html> <body> <?php ECHO "Hello World!<br>"; echo "Hello World!<br>"; EcHo "Hello World!<br>"; ?> </body> </html>

Sin embargo; todos los nombres de las variables entre mayúsculas y minúsculas. En el siguiente ejemplo, sólo la primera sentencia visualiza el valor de la variable $ color (esto se debe a $ color, $ color, y $ color son tratados como tres variables diferentes):

<!DOCTYPE html> <html> <body> <?php echo "My car is " . $color ."<br>"; echo "My car is " . $COLOR ."<br>"; echo "My car is " . $coLOR ."<br>"; ?> </body> </html>
