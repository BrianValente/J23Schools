El bucle for...in es usado para buclear entre las propiedades de un objeto. Como todavia no discutimos temas sobre objetos, puede que todavia no te sientas comodo con este bucle. Seguramente, cuando comprendas como los objetos se comportan en Javascript, te resulte util.

Sintaxis

for (nombreDeVariable en el objeto){ comando(s) a ejecutar; }

En cada iteracion, una propiedad del objeto es asignada a nombreDeVAriable y este bucle continua hasta que todas las propiedades del objeto son usadas.

Ejemplo.

Trata de ejecutar el siguiente ejemplo para implementar "for-in". Este, muestra los objetos del navegador web.
