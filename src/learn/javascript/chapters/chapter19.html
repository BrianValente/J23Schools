Capítulo 19: Impresión


Muchas veces se desea colocar un botón en su página web para imprimir el contenido de esa página web a través de una impresora real. JavaScript ayuda a implementar esta funcionalidad utilizando la función de impresión.


La función de impresión JavaScript window.print() imprime la página web actual cuando se ejecuta. Puede llamar directamente a esta función mediante el evento onclick, como se muestra en el siguiente ejemplo:


<!DOCTYPE html>

<html>
<head>
<script type="text/javascript"> </script>
</head>
<body>
<form>
<input type="button" value="Imprimir" onclick="window.print()" />
</form>
</body>
</html>

A pesar de que sirve al propósito de conseguir una impresión, no es un método recomendado. Una página impresa es en realidad sólo una página con el texto, sin imágenes, gráficos, o publicidad.


Puede hacer que una impresora funcione de la siguiente manera:


  • Haga una copia de la página y deje de lado el texto y los gráficos no deseados, a continuación, enlace esa página impresa a la original.
  • Si no desea conservar una copia adicional de una página, entonces usted puede marcar su texto imprimible usando los comentarios apropiados como ..... y luego se puede utilizar PERL o cualquier otro script en el fondo para purgar el texto imprimible y mostrar la impresión final.
