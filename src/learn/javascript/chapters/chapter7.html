Capítulo 7: Estructuras If y Else

Mientras escribe un programa, puede haber una situación en la que es necesario adoptar uno de un conjunto dado de caminos. En tales casos, es necesario utilizar instrucciones condicionales que permiten su programa para tomar decisiones correctas y realizar acciones correctas. JavaScript soporta sentencias condicionales que se utilizan para llevar a cabo diferentes acciones basadas en diferentes condiciones. Aquí vamos a explicar el if..else comunicado.

      if ... else

      El 'if ... else' declaración es el siguiente formulario de la declaración de control que permite JavaScript para ejecutar sentencias de una manera más controlada.

      Sintaxis

      if (expression){
      Statement(s) to be executed if expression is true
      }
      else{
      Statement(s) to be executed if expression is false
      }

      Aquí se evalúa JavaScript expresión. Si el valor resultante es cierto, la declaración (s) dada en el "si" de bloque, se ejecutan. Si la expresión es falsa, entonces la sentencia (s) dada en el bloque else se ejecuta.
      Ejemplo:
      Pruebe el siguiente código para aprender cómo implementar una instrucción if-else en JavaScript.

      <!--
      var age = 15;
      if( age > 18 ){
      document.write("Qualifies for driving");
      }
      else{
      document.write("Does not qualify for driving");
      }
      //-->
      </script>

      if ... else if ... comunicado

      El if ... else if ... afirmación es una forma avanzada de si ... otra cosa que permite JavaScript para tomar una decisión correcta de varias condiciones.

      Sintaxis

      La sintaxis de una instrucción if-else-if es la siguiente:
      if (expression 1){
      Statement(s) to be executed if expression 1 is true
      }
      else if (expression 2){
      Statement(s) to be executed if expression 2 is true
      }
      else if (expression 3){
      Statement(s) to be executed if expression 3 is true
      }
      else{
      Statement(s) to be executed if no expression is true
      }

      No hay nada especial en este código. Es sólo una serie de si los estados, donde cada caso es una parte de la otra cláusula de la declaración anterior. Declaración (s) se ejecuta sobre la base de la verdadera condición, si ninguna de las condiciones es verdadera, entonces el otrose ejecuta el bloque. Ejemplo: Pruebe el siguiente código para aprender cómo implementar una sentencia if-else-if en JavaScript.

      <html>
      <body>
      <script type="text/javascript">
      <!--
      var book = "maths";
      if( book == "history" ){
      document.write("History Book");
      }
      else if( book == "maths" ){
      document.write("Maths Book");
      }
      else if( book == "economics" ){
      document.write("Economics Book");
      }
      else{
      document.write("Unknown Book");
      }
      //-->
      </script>
