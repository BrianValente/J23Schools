Capítulo 4: Ubicar en el HTML

Hay cierta flexibilidad para incluir el código JavaScript en cualquier parte del código HTML. Hay lugares preferenciales, como son:

  • Script en la sección <head>…</head>
  • Script en la sección <body>…</body>
  • Script en ambas secciones
  • Script en un archivo externo, incluido en la sección <head>…</head>.

Script en <head>…<head>

Generalmente, estos scripts se colocan cuando van a ser ejecutados por un evento, como un click en algún botón o imagen.

<html>
<head>
<script type="text/javascript">
<!--
function sayHello() {
alert("Hello World")
}
//-->
</script>
</head>
<body>
<input type="button" onclick="sayHello()" value="Say Hello" /> </body>
</html>

JavaScript en <body>…</body>

Si necesitás que el script se ejecute cuando la página carga, y genere de esta manera contenido en la misma, entonces el script debe ir en <body>…</body>.

<html>
<head> </head>
<body>
<script type="text/javascript">
<!--
document.write("Hello World")
//-->
</script>
<p>This is web page body </p>
</body>
</html>

Script en <head> y en <body>

Hay casos en los que se necesita que un script se ejecute con la carga de la página, y otro que se ejecute sólo con un click. Es entonces cuando se coloca el script en ambas secciones del documento.

<html>
<head>
<script type="text/javascript">
<!-- function sayHello() { alert("Hello World") } //-->
</script>
</head>
<body>
<script type="text/javascript">
<!--
document.write("Hello World")
//-->
</script>
<input type="button" onclick="sayHello()" value="Say Hello" />
</body>
</html>

JavaScript en un archivo externo

Hay casos en los que un script debe utilizarse una y otra vez en diferentes páginas. Si bien nada te impide el escribirlo cada vez que lo necesites, es más conveniente tenerlo en un archivo *.js que luego será llamado desde el documento HTML, como si se tratara de un stylesheet de CSS.

 <head>
   <script type="text/javascript" src="filename.js" ></script>
  </head>

Cuando usás un script que está escrito en un archivo externo, podés llamar las funciones del mismo en cualquier parte de tu código, siguiendo los casos que explicamos anteriormente. Esto luciría de la siguiente manera, teniendo por ejemplo un archivo con una función en su interior llamada sayHello().

 function sayHello() {   alert("Hello World") }
