Capítulo 9: Ciclo while

Mientras escribe un programa, puede encontrarse con una situación en la que necesita para realizar una acción una y otra vez. En tales situaciones, lo que se necesita para escribir sentencias de bucle para reducir el número de líneas.
JavaScript soporta todos los bucles necesarios para aliviar abajo la presión de la programación.

El bucle while

El bucle más básico en JavaScript es el tiempo de bucle que se describen en este capítulo. El propósito de un tiempo de bucle es ejecutar una instrucción o bloque de código varias veces, siempre y cuando una expresión es verdadera. Una vez que la expresión se convierte en falsa, el bucle termina.

Diagrama de flujo El diagrama de flujo del bucle while se ve de la siguiente manera:

Sintaxis: La sintaxis del bucle while en JavaScript es el siguiente:

while (expression){
Statement(s) to be executed if expression is true
}

El bucle do ... while

El tiempo hacer ... bucle es similar al tiempo de bucle excepto que la comprobación de condición que ocurre al final del bucle. Esto significa que el bucle siempre se ejecuta al menos una vez, incluso si la condición es falsa . Diagrama de flujo El diagrama de flujo de un do-while bucle sería de la siguiente manera:


Do While Loop

Sintaxis: La sintaxis para do-while bucle en JavaScript es el siguiente:

do{
Statement(s) to be executed;
} while (expression);

Nota - No se pierda el punto y coma se utiliza al final del bucle do ... while. Ejemplo Prueba el siguiente ejemplo para aprender cómo implementar un do-while bucle en JavaScript.
