Capítulo 16: Redirección de páginas


¿Qué es la redirección de la página?


Es posible que haya encontrado con una situación en la que ha hecho clic en un enlace para llegar a una página X pero internamente se le ha remitido a otra página Y. Esto ocurre debido a la página de redirección. Este concepto es diferente de JavaScript Page Refresh.


Podría haber varias razones por las que le gustaría volver a dirigir a un usuario la página original. Esta lista muestra algunas de las razones:

  • No te gustaba el nombre de su dominio y se está moviendo a una nueva. En tal escenario, es posible que desee dirigir todos sus visitantes al nuevo sitio. Aquí usted puede mantener su dominio de edad pero puso una sola página con una redirección de la página de tal manera que todos sus viejos visitantes de dominio puedan llegar a su nuevo dominio.
  • Usted ha construido en marcha varias páginas basadas en versiones de navegadores o sus nombres o puede basarse en diferentes países, a continuación, en lugar de utilizar la página en el servidor de redirección, puede utilizar la redirección de la página del lado del cliente a sus usuarios de la tierra en la página correspondiente.
  • Los motores de búsqueda pueden ya han indexado sus páginas. Pero mientras se mueve a otro dominio, no le gustaría perder a sus visitantes a través de motores de búsqueda. Así que usted puede utilizar la página de redirección del lado del cliente. Pero hay que tener en cuenta que no se debe hacer para engañar a los motores de búsqueda, que podría conseguir que sea baneado.

¿Cómo funciona la Re-dirección?


Las implementaciones de la página-redirección son las siguientes:


Ejemplo 1

Es muy simple hacer una redirección de una página usando JavaScript del lado del cliente. Para redirigir los visitantes del sitio a una nueva página, sólo tiene que añadir una línea en su sección de la cabeza de la siguiente manera:


<!DOCTYPE html>

<html>
<head>
<script type="text/javascript">
function Redirect() {
window.location="j23schools.ml";
}
</script>
</head>
<body>
<form>
<input type="button" value="Me redirecciona" onclick="Redirect();" />
</form>
</body>
</html>

Ejemplo 2

Puede mostrar un mensaje adecuado a sus visitantes del sitio antes de redirigir a una nueva página. Esto necesitaría un retardo de poco tiempo en cargar una página nueva. El siguiente ejemplo muestra cómo implementar la misma. Aquí setTimeout() es una función incorporada en JavaScript que puede ser utilizada para ejecutar otra función después de un intervalo de tiempo dado.


<!DOCTYPE html>

<html>
<head>
<script type="text/javascript">
function Redirect() {
window.location="j23schools.ml";
}
document.write("Usted será redirigido a la página principal en 10 seg.");
setTimeout('Redirect()', 10000);
</script>
</head>
</html>

Ejemplo 3

El siguiente ejemplo muestra cómo redirigir a sus visitantes del sitio a una página diferente en función de sus navegadores.


<!DOCTYPE html>

<html>
<head>
<script type="text/javascript">
var browsername=navigator.appName;
if( browsername == "Netscape" ) {
window.location="http://www.location.com/ns.htm";
}
else if ( browsername =="Microsoft Internet Explorer") {
window.location="http://www.location.com/ie.htm";
}
else {
window.location="http://www.location.com/other.htm";
}
</script>
</head>
</html>
