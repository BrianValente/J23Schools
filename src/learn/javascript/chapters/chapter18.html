Capítulo 18: Palabras claves: void


Void es una palabra clave importante en JavaScript que puede ser utilizada como un operador unitario que aparece antes de un operador único, que puede ser de cualquier tipo. Este operador especifica una expresión a ser evaluada sin devolver un valor.


Sintaxis


La sintaxis de un void puede ser cualquiera de los dos siguientes:


<!DOCTYPE html>

<html>
<head>
<script type="text/javascript">

<!--
void func()
javascript:void func()
o:
void(func())
javascript:void(func())
//-->

</script>
</head>
</html>

Ejemplo 1


El uso más común de este operador está en un client-side (lado del cliente) javascript: URL, donde se le permite evaluar una expresión para sus side-effects (efectos secundarios) sin el explorador que muestra el valor de la expresión evaluada.


Aquí la expresión alert ( 'Advertencia!!!') se evaluó pero no se carga de nuevo en el documento actual.


<!DOCTYPE html>

<html>
<head>
<script type="text/javascript"> </script>
</head>
<body>
<p>Haga clic en el siguiente, Esto no va a reaccionar en absoluto...</p>
<a href="javascript:void(alert('Advertencia!!!'))">Haz clic en mi!</a>
</body>
</html>

Ejemplo 2


El siguiente enlace no hace nada porque la expresión "0" no tiene ningún efecto en JavaScript. Aquí se evalúa la expresión "0", pero no se carga de nuevo en el documento actual.


<!DOCTYPE html>

<html>
<head>
<script type="text/javascript"> </script>
</head>
<body>
<p>Haga clic en el siguiente, Esto no va a reaccionar en absoluto...</p>
<a href="javascript:void(0)">Haz clic en mi!</a>
</body>
</html>

Ejemplo 3


Otro uso del void es generar deliberadamente el valor indefinido de la siguiente manera:


<!DOCTYPE html>

<html>
<head>
<script type="text/javascript">
function getValue() {
var a,b,c;
a = void (b = 5, c = 7);
document.write('a = ' + a + ' b = ' + b +' c = ' + c );
}
</script>
</head>
<body>
<p>Haga clic en el siguiente para ver el resultado:</p>
<form>
<input type="button" value="Haz clic en mi" onclick="getValue();" />
</form>
</body>
</html>
