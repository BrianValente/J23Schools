JavaScript provee maneras de controlar completamente los bucles y switches. Puede que haya situaciones en las que necesites salir de un bucle antes de terminarlo, y otras en las que quiza quieras saltear alguna parte de tu bloque de comandos y saltar a la proxima iteracion del bucle.
Para manejar estas situaciones, JavaScript tiene los comandos "break" y "continue". Estos, son usados para salir del bucle o saltar a la proxima iteracion respectivamente.

Break.

El comando break, que en un comienzo fue introducido junto al comando switch, es usado para salir de los bucles antes de terminar, rompiendo asi el circulo.

  • Diagrama de Flujo.

    El diagrama de flujo para el comando "break" podria interpretarse como el siguiente. >

  • Ejemplo.

    El siguiente ejemplo muestra el uso del comando break para salir del bucle antes de que se termine de ejecutar.

    <html> <body> <script type="text/javascript"> var x = 1; document.write("Entering the loop
    "); while (x < 20) { if (x == 5){ break; // breaks out of loop completely } x = x + 1; document.write( x + "<br />"); } document.write("Exiting the loop<br /> "); </script> <p>Set the variable to different value and then try...</p> </body> </html>

  • Output

    Entering the loop 2 3 4 5 Exiting the loop! Set the variable to different value and then try...

Continue

El comando "continue" hace que la iteracion termine de inmediato y no ejecute los comandos que restan ejecutarse. Cuando se encuentra un "continue", se verifica si la condicion sigue siendo verdadera, de ser asi continua con el bucle. Caso contrario lo finaliza y sale del bucle.

    Ejemplo.

    Este ejemplo demuestra el funcionamiento de un comando "continue" en un bucle "while". Podes notar como al estar el "continue", no se ejecuta el bloque de comandos restantes y salta a la siguiente iteracion.

    <html> <body> <script type="text/javascript"> var x = 1; document.write("Entering the loop<br /> "); while (x < 10) { x = x + 1; if (x == 5){ continue; // skip rest of the loop body } document.write( x + "<br />"); } document.write("Exiting the loop!<br /> "); </script> <p>Set the variable to different value and then try...</p> </body> </html>

  • Output

    Entering the loop 2 3 4 6 7 8 9 10 Exiting the loop!

Usando etiquetas para controlar el flujo

A partir de JavaScript 1.2, una etiqueta puede ser usada junto a "break" y "continue" para controlar aun mejor el flujo del codigo.
Una etiqueta se declara a partir de una palabra clave o identificador seguido de un dos puntos (:) que se aplica a uno o varios comando de un codigo.
Hay que tener en cuenta que los saltos de linea no se pueden usar entre "continue" o "break" y su etiqueta. Ademas, no deberian repetirse los nombres ni entre etiquetas ni bucles asociados.
Prueba el siguiente ejemplo para entender mejor el uso y manejo de etiquetas:

Ejemplo 1

Este ejemplo demuestra el uso de etiquetas junto con "break"s.

<html> <body> <script type="text/javascript"> document.write("Entering the loop!<br /> "); outerloop: // This is the label name for (var i = 0; i < 5; i++) { document.write("Outerloop: " + i + "<br />"); innerloop: for (var j = 0; j < 5; j++) { if (j > 3 ) break ; // Quit the innermost loop if (i == 2) break innerloop; // Do the same thing if (i == 4) break outerloop; // Quit the outer loop document.write("Innerloop: " + j + " <br />"); } } document.write("Exiting the loop!<br /> "); </script> </body> </html>

Output
