Capítulo 13: Funciones

  • Una funcion es un bloque de codigo reusable que puede ser llamado todas las veces que sea necesario.
  • Esto evita el uso excesivo de lineas y la re-escritura de una misma parte del codigo, ayuda a los programadores a escribir codigos modulares
  • Las funciones permiten al programador a separar un gran programa en muchas programas mas pequeños, haciendolo mas manejable y flexible.

Como cualquier otro lenguaje de programacion avanzado, JavaScript tambien soporta todas las caracteristicas necesarias para escribir codigos modulares. Seguramente hayas visto funciones como alert() y write() en capitulos anteriores. Como habras notado, usamos esas funciones varias veces, pero solo fueron escritas en el nucleo de JavaScript una sola vez.

  • JavaScript nos permite escribir nuestras propias funciones. Esto es lo que vamos a ver en este capitulo.

Definicion de funciones

Antes de usasr una funcion, debemos definirla. La manera mas comun de hacerla en JavaScript es usando la palabra clave "function", seguido del nombre unico de la funcion, una lista de parametros(o ninguno), y una secuencia de comandos para relizar cuando se llame a la funcion.

Sintaxis

La sintaxis basica de una funcion es la siguiente:
<script type="text/javascript"> function functionname(parameter-list) { statements } </script>

Ejemplo

Trata de hacer el siguiente ejemplo. Es una funcion llamada sayHello que no toma parametros de ningun lado.

<script type="text/javascript"> function sayHello() { alert("Hello there"); } </script>

Llamando a la funcion

Para llamar mas tarde a la funcion en el qcodigo, simplemente necesitas escribir el nombre de esa funcion, asi como se demuestra en el siguiente codigo.

<html> <head> <script type="text/javascript"> function sayHello() { document.write ("Hello there!"); } </script> </head> <body> <p>Click the following button to call the function</p> <form> <input type="button" onclick="sayHello()" value="Say Hello"> </form> <p>Use different text in write method and then try...</p> </body> </html>

Parametros de una funcion

Hasta ahora, solo habiamos visto funciones sin parametros. Ahora vamos a ver como hacer esto, capturar parametros en la funcion mientras la llamamos. Estos parametros podemos manipularlos libremente una vez capturados. Una misma funcion puede capturar multiples parametros, cada uno debe estar separado por una coma.

Ejemplo

En el siguiente ejemplo modificamos un poco la funcion sayHello que utilizamos antes y ahora toma dos parametros.

<html> <head> <script type="text/javascript"> function sayHello(name, age) { document.write (name + " is " + age + " years old."); } </script> </head> <body> <p>Click the following button to call the function</p> <form> <input type="button" onclick="sayHello('Zara', 7)" value="Say Hello"> </form> <p>Use different parameters inside the function and then try...</p> </body> </html>

El retorno de la funcion.

Una funcion de JavaScript puede tener un retorno opcional. Esto puede requerido si se quiere devolver algun valor al codigo original. ESte ademas, debe ser declarado al final de todo el bloque.
Por Ejemplo, se puede crear una funcion que pida dos parametros numericos, cuyo retorno seria su multiplicacion o cualquier accion.

Ejemplo

En el siguiente ejemplo vamos a poner en pratica lo dicho anteriormente. Vamos a tomar dos palabras y las vamos a concatenar en una sola.

<html> <head> <script type="text/javascript"> function concatenate(first, last) { var full; full = first + last; return full; } function secondFunction() { var result; result = concatenate('Zara', 'Ali'); document.write (result ); } </script> </head> <body> <p>Click the following button to call the function</p> <form> <input type="button" onclick="secondFunction()" value="Call Function"> </form> <p>Use different parameters inside the function and then try...</p> </body> </html>
